Calculator V2 still a C# Console App

Overview
I enhanced my original calculator by improving its structure and adding more functionality. This version introduces better control flow, error handling, and a more user-friendly experience.

---

Features
- Addition, subtraction, multiplication, and division
- Modulus (%) operation
- Continuous loop (allows multiple calculations)
- Switch-case for cleaner and more readable logic
- Error handling using try-catch
- Input validation (e.g., division by zero)
- Exit option

---

Technologies Used
- C#
- .NET Console Application

---

What I Learned
- Using loops to keep applications running continuously
- Implementing switch statements for cleaner logic
- Handling errors with try-catch blocks
- Improving code readability and structure
- Building more user-friendly console applications
